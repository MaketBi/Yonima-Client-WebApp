# Icônes PWA requises pour Yonima

## Icônes principales (à générer)
Les fichiers suivants doivent être créés avec le logo Yonima sur fond #1DB88E :

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Icônes maskable (Android adaptive icons)
Ces icônes doivent avoir un padding de 20% pour la safe zone :

- icon-maskable-192x192.png
- icon-maskable-512x512.png

## Icônes shortcuts
Pour les raccourcis de l'écran d'accueil :

- shortcut-restaurant.png (96x96) - Icône restaurant
- shortcut-orders.png (96x96) - Icône commandes
- shortcut-cart.png (96x96) - Icône panier

## Screenshots
Pour l'installation PWA :

- screenshots/home.png (390x844) - Capture d'écran de l'accueil

## Comment générer les icônes

1. Créer le logo Yonima en SVG ou PNG haute résolution (au moins 512x512)
2. Utiliser un outil comme https://realfavicongenerator.net/ ou https://www.pwabuilder.com/imageGenerator
3. Ou utiliser ImageMagick :
   ```bash
   # Depuis un fichier source icon-source.png (512x512 ou plus)
   convert icon-source.png -resize 72x72 icon-72x72.png
   convert icon-source.png -resize 96x96 icon-96x96.png
   convert icon-source.png -resize 128x128 icon-128x128.png
   convert icon-source.png -resize 144x144 icon-144x144.png
   convert icon-source.png -resize 152x152 icon-152x152.png
   convert icon-source.png -resize 192x192 icon-192x192.png
   convert icon-source.png -resize 384x384 icon-384x384.png
   convert icon-source.png -resize 512x512 icon-512x512.png
   ```

## Couleurs Yonima
- Couleur primaire : #1DB88E (teal/vert)
- Fond clair : #FFFFFF
- Fond sombre : #0F172A
